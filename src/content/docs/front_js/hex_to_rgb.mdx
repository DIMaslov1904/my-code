title: Hex в RGB
---

`hexToRgb(color)` -
Передаем цвет в формате hex (#f2f2f2), получаем цвет в формате rgb (rgb (123, 123, 123)) 

```ts
export function hexToRgb(hex: string) {
  hex = hex.replace("#", "");
  const bigint = parseInt(hex, 16);
  const r = (bigint >> 16) & 255;
  const g = (bigint >> 8) & 255;
  const b = bigint & 255;
  return [r, g, b].join();
}
```
